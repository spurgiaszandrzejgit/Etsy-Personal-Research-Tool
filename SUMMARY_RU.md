## 📊 Что умеет приложение

### Анализирует ниши Etsy за 4 секунды:
1. **Цены**: среднее, медиана, мин/макс, распределение
2. **Конкуренция**: Low/Medium/High (< 50 / 50-200 / 200+)
3. **Ключевые слова**: частота, топ-20, теги
4. **Магазины**: топ конкуренты, доминирование
5. **Оценка ниши**: 1-10 баллов (многофакторная)

### Генерирует Excel отчёты:
- **Summary**: KPI, метрики, оценка ниши
- **Listings**: все товары (до 200)
- **Shops**: топ конкуренты с рейтингом
- **Keywords**: частота ключевых слов

## 🎯 Как использовать

### ⚠️ ВАЖНО: Сначала настройте API ключ!

**Если вы видите ошибку "404 Not Found":**
1. 📖 Откройте **[API_KEY_SETUP.md](API_KEY_SETUP.md)** - пошаговая инструкция
2. Получите бесплатный API ключ на https://www.etsy.com/developers/
3. Добавьте его в `appsettings.json`
4. Запустите приложение снова

### Быстрый старт (5 минут):
```bash
# 1. Получите API ключ: https://www.etsy.com/developers/
# 2. Откройте: src/EtsyAnalyzer.ConsoleApp/appsettings.json
# 3. Замените YOUR_ETSY_API_KEY_HERE на ваш ключ
# 4. Запустите:
cd src/EtsyAnalyzer.ConsoleApp
dotnet run
```

### Первый анализ:
1. Выберите "Analyze New Niche"
2. Введите: `handmade soap`
3. Подождите ~5 секунд
4. Просмотрите результаты + Excel отчёт
